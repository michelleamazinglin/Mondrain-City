1. Users cam add an Apartment with a name, creates a blue block on canvas.

  > <img width="295" alt="Screen Shot 2021-05-13 at 11 53 53 AM" src="https://user-images.githubusercontent.com/68549082/118151694-e53e2200-b3e1-11eb-9398-06b3ecaf8ffc.png">

  > <img width="257" alt="Screen Shot 2021-05-13 at 11 54 06 AM" src="https://user-images.githubusercontent.com/68549082/118151729-ecfdc680-b3e1-11eb-9494-1680867513cc.png">
